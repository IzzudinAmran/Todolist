1. git cloned repo https://github.com/IzzudinAmran/todolist.git
2. copy env.example file and create new .env file
3. change the DB_DATABASE=todolist, DB_USERNAME=root , DB_PASSWORD=' '
4. run composer install/update
5. run php artisan key:generate to generate cypher key
6. run php artisan migrate
7. run php artisan serve
